# qDrive0's Portfolio

## Getting Started
Instructions assume that you are running macOS.

Install task:

`brew install go-task`

You can enable autocompletions by setting the following lines in your `~/.zshrc`

```
# Top of file
autoload -Uz compinit
compinit

# export TASK_EXE='go-task' if needed
eval "$(task --completion zsh)"
```

To run static Jekyll site locally,

`task run_jekyll`
